# sismerenda
É um software desenvolvido pelo curso de informática da Etec Zona
Leste para gerenciamento de merenda escolar.

# Para usar este projeto
Siga esse passo a passo. Depois os professores do curso publicam
tutoriais com detalhes de cada passo, por enquanto usem o google ou
pergunte a um dos professores de programação do curso.

1. Instale o git na sua máquina
2. Instale o java 
3. Instale o maven
4. Clone este repositório com o seguinte comando
   *git clone https://github.com/SolucoesemTI/sismerenda.git*
   
   Esse comando vai criar um diretório do projeto na sua máquina.
   
5. Entre no diretório do projeto 
6. Pra rodar o projeto você vai dar dois comandos
   
   *mvn clean package*
   Este comando vai limpar e contruir seu projeto e criar um jar no
   diretório target, dentro do diretório do seu projeto
   
   *java -jar target/sismerenda-1.0-SNAPSHOT.jar*
